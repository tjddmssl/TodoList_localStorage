## Features

- 편리한 UI로 Todo를 쉽게 생성하고 삭제할 수 있습니다.
- 완료한 TodoList와 아직 완료하지 못한 TodoList를 나눠서 관리할수 있습니다.
- create-react-app으로 간편한 번들링과 배포가 가능합니다.
- localStorage를 사용하여 데이터를 GET,POST,DELETE,PATCH하였습니다.
